# On-Page SEO Plan - Lastframe.video

## Goal
Rank for high-intent queries around final frame extraction and video frame download in US and UK markets.

## Primary keyword
- final frame extract

## Secondary keyword clusters
- Brand: final frame, finalframe, lastframe
- Core action: final frame extractor, last frame extractor, video frame extractor, frame extractor from video
- Task intent: extract last frame of video, get last frame of video, get frame from video, capture frame from video
- Tool intent: video frame downloader, frame grabber, mp4 frame extractor, video to frames
- AI continuity intent: frame ai video, start and end frame ai video, first frame to last frame ai

## On-page implementation
- Title updated with primary keyword and brand.
- Meta description rewritten for intent + geo context (US/UK).
- Canonical set to production domain.
- Open Graph and Twitter metadata updated.
- FAQ schema added for rich results eligibility.
- SoftwareApplication schema updated with free offer and tool description.
- H1 aligned to primary query intent.
- Added semantic sections for:
  - How-to steps
  - Use cases
  - FAQ content
- Added naturally distributed keyword variants in body copy.

## Content rules for future updates
- Keep one clear H1 only.
- Keep keyword usage natural; avoid repeating the same exact phrase excessively.
- Add one new FAQ per week based on search console queries.
- Keep copy aligned with actual tool behavior (last 5 frames, click to download).

## Execution workflow
1. Implement changes on `codex/ui-preview-1`.
2. Review Netlify Deploy Preview.
3. Merge PR to `main` only after approval.
